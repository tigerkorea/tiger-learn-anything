## 게시물 1 — 훅 + 핵심 설정

```
어떤 주제든 AI가 주차별 공부 자료를 만들어줍니다 📚

비트코인, 마라톤, 현대사, Python...
주제를 말하면 4가지만 물어봐요:

  📊 난이도?
     입문 / 중급 / 고급

  📅 기간?
     4주 / 8주 / 12주

  🤖 모델?
     Sonnet(빠름) / Opus(고품질)

  📁 저장 폴더?
     기본값 or 직접 입력

그러면 커리큘럼 + 주차별 학습 자료를
마크다운 파일로 자동 생성해줘요
```

## 게시물 2 — 결과물 상세

```
각 주차 파일에 이게 다 들어가요 👇

🧩 핵심 개념 (정의 + 예시 2개 + 비유)
📊 알아두면 좋은 데이터
🔗 추천 학습 자료 (링크 포함)
🛠️ 실습 과제 (단계별 + 완료 조건)
✅ 주간 체크리스트
❓ 퀴즈 5문항 + 정답
🚀 다음 주 예고

1주차 생성 후 형식 피드백도 받아요
"설명 더 길게", "퀴즈 10개로" 같이 말하면
스킬이 스스로 규칙을 업데이트하고 재생성

Claude Code 스킬이라 설치하면
어느 프로젝트에서든 바로 쓸 수 있어요
```
